# Clue/Cluedo Assistant - Feature Recommendations

## High Priority Features

### 1. **Auto-Save on Every Action**
- Automatically save game state after every card mark, suggestion, etc.
- No need to manually click "Save Game"
- Prevents data loss

### 2. **Export/Import Game State**
- Export game state as JSON file
- Import game state from file
- Share game state with other players
- Backup/restore functionality

### 3. **Card Count Tracker**
- Show how many cards each player should have based on number of players
- Alert when a player has too many/few cards marked
- Helps catch mistakes

### 4. **Probability Calculator**
- Calculate probability that each card is in the envelope
- Show percentages based on known information
- Help prioritize which cards to investigate

### 5. **Quick Mark Actions**
- Mark multiple cards at once (e.g., "Player 1 doesn't have any of these 3 cards")
- Bulk operations for efficiency
- Quick filters for cards

## Medium Priority Features

### 6. **Notes Section**
- Add notes for each player
- Track suspicions, observations
- Personal reminders

### 7. **Turn Order Indicator**
- Visual indicator of whose turn it is
- Track current turn in the game
- Next player indicator

### 8. **Enhanced Undo/Redo**
- Full undo/redo stack (not just last move)
- History navigation
- Multiple steps back

### 9. **Card Filtering/Search**
- Filter cards by category
- Search for specific cards
- Show only unknown cards
- Show only cards in envelope

### 10. **Statistics Dashboard**
- Cards discovered count
- Suggestions made
- Deductions found
- Game progress percentage

## Nice-to-Have Features

### 11. **Dark Mode**
- Toggle between light/dark themes
- Better for playing in low light
- Eye strain reduction

### 12. **Sound Effects/Notifications**
- Audio feedback for actions
- Notification when deduction is made
- Optional sound toggle

### 13. **Timer for Turns**
- Optional timer for each player's turn
- Helps keep game moving
- Configurable time limits

### 14. **Share Game State**
- Generate shareable link
- QR code for easy sharing
- Multi-device sync

### 15. **Card Images**
- Visual card representations
- Classic Clue card designs
- Better visual recognition

### 16. **Accusation Tracker**
- Track when players make accusations
- Record failed accusations
- Help narrow down possibilities

### 17. **Player Color Coding**
- Assign colors to players
- Visual distinction in grid
- Easier to track

### 18. **Keyboard Shortcuts**
- Quick actions via keyboard
- Faster data entry
- Power user features

### 19. **Game Statistics**
- Win/loss tracking
- Average game length
- Most common solutions
- Personal bests

### 20. **Tutorial/Help Mode**
- Interactive tutorial for new users
- Tooltips explaining features
- Example game walkthrough

## Mobile-Specific Enhancements

### 21. **Swipe Gestures**
- Swipe to mark cards
- Swipe to undo
- Gesture-based navigation

### 22. **Haptic Feedback**
- Vibration on actions
- Tactile confirmation
- Better mobile UX

### 23. **Offline Mode**
- Works without internet
- PWA (Progressive Web App) support
- Install to home screen

### 24. **Landscape Mode Optimization**
- Better layout for landscape
- More cards visible
- Optimized for tablets

## Implementation Priority

**Start with:**
1. Auto-save (easy, high value)
2. Card count tracker (medium effort, high value)
3. Export/Import (medium effort, high value)

**Then add:**
4. Probability calculator (higher effort, high value)
5. Notes section (easy, medium value)
6. Enhanced undo/redo (medium effort, medium value)

**Polish with:**
7. Dark mode (easy, nice-to-have)
8. Statistics (medium effort, nice-to-have)
9. Keyboard shortcuts (medium effort, nice-to-have)
